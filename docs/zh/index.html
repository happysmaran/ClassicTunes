ClassicTunes

带回经典的 iTunes 体验

关于

ClassicTunes 是一款从零开始使用 Swift 和 SwiftUI 构建的现代 macOS 应用程序，在现代 Apple Silicon Mac 上忠实再现了 iTunes 7-10 的外观和体验。

  • 可用的 Cover Flow
  • 迷你播放器 (MiniPlayer)
  • 播放列表（支持智能 m3u）
  • iTunes Store 集成
  • 歌词支持（使用 lrclib 或元数据）
  • iPod 同步 (1st-2nd gen Shuffles)
  • Discord Rich Presence (集成 MusicPresence)
  • 经典的 iTunes 7-10 美学

系统要求：macOS 13 Ventura 或更高版本，Apple Silicon (M1 及更高版本)。
