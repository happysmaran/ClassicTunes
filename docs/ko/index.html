ClassicTunes

클래식 iTunes 경험의 귀환

정보

ClassicTunes는 Swift 및 SwiftUI를 사용하여 처음부터 구축된 최신 macOS 앱으로, 최신 Apple Silicon Mac에서 iTunes 7~10의 모양과 느낌을 충실하게 재현합니다.

  • 작동하는 Cover Flow
  • 미니 플레이어
  • 재생 목록 (스마트 m3u 지원 포함)
  • iTunes Store 통합
  • 가사 지원 (lrclib 또는 메타데이터 사용)
  • 아이팟 싱킹 (1st-2nd gen Shuffles)
  • Discord Rich Presence (MusicPresence 사용)
  • 클래식 iTunes 7-10 미학

시스템 요구 사항: macOS 13 Ventura 이상, Apple Silicon (M1 이상).
